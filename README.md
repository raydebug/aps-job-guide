# APS 找工指导网站

一个只聚焦 APS 求职的静态指南网站，重点是 merit evidence、500-word pitch、selection criteria、APS 等级判断、panel interview 和 referee validation。

## 本地预览

```bash
npm install
npm run dev
```

## 发布到 GitHub Pages

1. 在 GitHub 新建仓库。
2. 把本项目推送到 `main` 分支。
3. 在仓库设置中打开 `Pages`，将发布来源设为 `GitHub Actions`。
4. 每次推送到 `main` 后会自动部署。
